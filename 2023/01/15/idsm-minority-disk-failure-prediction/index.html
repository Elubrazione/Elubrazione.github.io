дҪҝз”ЁиҝҒз§»еӯҰд№ 

в‘  еӨ§еһӢеӯҳеӮЁзі»з»ҹдёӯзЈҒзӣҳж•…йҡңдјҡеҜјиҮҙдёҘйҮҚж•°жҚ®дёўеӨұз”ҡиҮійҡҫд»ҘеӨҚеҺҹж•°жҚ®пјӣв‘Ў еҪ“еүҚдё»жөҒж–№жі•иҝҳжңүreplicationе’Ңerasure codingзӯүжҒўеӨҚж•°жҚ®жҠҖжңҜпјҢдҪҶзЈҒзӣҳж•…йҡңйў„жөӢжҠҖжңҜдёҚд»…еҸҜд»ҘйҷҚдҪҺдёўеӨұж•°жҚ®зҡ„в»ӣйҷ©пјҢиҝҳеҸҜд»ҘйҷҚдҪҺж•°жҚ®жҒўеӨҚжҲҗжң¬пјӣв‘ў з”ұдәҺзјәд№Ҹи¶іеӨҹе°‘ж•°зЈҒзӣҳзҡ„и®ӯз»ғж•°жҚ®пјҢйңҖиҰҒеӨ§йҮҸи®ӯз»ғж•°жҚ®зҡ„дј з»ҹMLж–№жі•е®№жҳ“еҸ‘з”ҹиҝҮжӢҹеҗҲпјҢж— жі•еұ•зӨәеҮәдјҳз§Җзҡ„йў„жөӢж•Ҳжһңпјӣ

дҪҝз”Ёзҡ„ж–№жі•

  1. дҪҝз”ЁиҝҒз§»еӯҰд№ д»ҺеҸҜз”Ёзҡ„еӨҡж•°зЈҒзӣҳж•°жҚ®йӣҶдёӯйў„жөӢе°‘ж•°зЈҒзӣҳзҡ„ж•…йҡң
  2. дҪҝз”ЁдёҖз§ҚеҹәдәҺKLDеҖјзҡ„ж–№жі•жқҘйҖүжӢ©еҗҲйҖӮзҡ„еӨҡж•°зЈҒзӣҳжЁЎеһӢ

дё»иҰҒеҶ…е®№

  1. е°‘ж•°зЈҒзӣҳзҡ„е®ҡд№үпјҡ1500дёәеҲҶж°ҙеІӯпјҲпјҹпјү

  2. дёәд»Җд№ҲдҪҝз”ЁиҝҒз§»еӯҰд№ 

    • жҜҸдёӘеҲ¶йҖ е•Ҷзҡ„дёҚеҗҢзЈҒзӣҳжЁЎеһӢзҡ„ж•…йҡңжЁЎејҸеҚҒеҲҶзӣёдјј

      вҮ’ еә”вҪӨиҝҒз§»еӯҰд№ жқҘиҝӣвҫҸи·ЁжЁЎеһӢзЈҒзӣҳж•…йҡңйў„жөӢ

    • еҗҢвјҖеҲ¶йҖ е•Ҷзҡ„дёҚеҗҢзЈҒзӣҳеһӢеҸ·иЎЁзҺ°еҮәдёҚеҗҢзҡ„SMARTеҖјеҲҶеёғ

      вҮ’ еҹәдәҺдј з»ҹ ML з®—жі•дҪҝвҪӨвјҖз§ҚзЈҒзӣҳжЁЎеһӢзҡ„и®ӯз»ғж•°жҚ®е»әвҪҙзҡ„йў„жөӢжЁЎеһӢдёҚйҖӮвҪӨдәҺе…¶д»–еҚідҪҝжқҘвҫғеҗҢвјҖеҲ¶йҖ е•Ҷзҡ„дёҚеҗҢжЁЎеһӢ

  3. еҰӮдҪ•дҪҝз”ЁиҝҒз§»еӯҰд№ пјҲз®—жі•з»ҶиҠӮпјү

  4. дҪ•ж—¶дҪҝз”ЁиҝҒз§»еӯҰд№ пјҲKLDзҡ„дҪҝз”Ёпјү

    иЎЎйҮҸвјҖдёӘжҰӮзҺҮеҲҶеёғдёҺеҸҰвјҖдёӘйў„жңҹжҰӮзҺҮеҲҶеёғзҡ„еҲҶжӯ§зЁӢеәҰзҡ„жҢҮж ҮпјҢиЎЁвҪ°дёӨдёӘйҡҸжңәеҸҳйҮҸеҲҶеёғд№Ӣй—ҙзҡ„е·®ејӮгҖӮ

    йӣ¶ KLD еҖјж„Ҹе‘ізқҖдёӨдёӘйҡҸжңәеҲҶеёғзӣёеҗҢпјҢKLD еҖјйҡҸзқҖдёӨдёӘйҡҸжңәеҲҶеёғд№Ӣй—ҙзҡ„е·®ејӮжү©вјӨвҪҪеўһеҠ гҖӮKLD еҖји¶ҠвјӨпјҢдёӨдёӘеҲҶеёғд№Ӣй—ҙзҡ„е·®ејӮи¶ҠвјӨпјҢдёӨдёӘеҲҶеёғд№Ӣй—ҙзҡ„иҝҒз§»е°ұи¶Ҡеӣ°йҡҫгҖӮ

е®һйӘҢпјҡж•°жҚ®йӣҶгҖҒеҜ№жҜ”зҡ„з®—жі•гҖҒдјҳи¶ҠжҖ§

ж•°жҚ®йӣҶ

Backblazeзҡ„SMARTејҖжәҗж•°жҚ®йӣҶе’Ңи…ҫи®Ҝж•°жҚ®дёӯвјјзҡ„ж•°жҚ®йӣҶгҖӮдҪҝз”Ёж¬ йҮҮж ·зұ»пјҢж•…йҡңпјҡиүҜеҘҪ = 1пјҡ3пјҢи®ӯз»ғйӣҶе’ҢжөӢиҜ•йӣҶ7пјҡ3пјҢдәӨеҸүйӘҢиҜҒгҖӮ

еҜ№жҜ”зҡ„з®—жі•

  1. дёҺдј з»ҹMLз®—жі•зӣёжҜ”иҫғ вҮ’ 3.3

    • FDRпјҡеӣӣз§Қдј з»ҹзҡ„ ML вҪ…жі•йғҪдёҚиғҪдҪҝвҪӨвјӨеһӢејӮжһ„ж•°жҚ®йӣҶжҸҗдҫӣвҫј FDRгҖӮвҪҪTLDFPеҲҶеҲ«дҪҝвҪӨGBRTгҖҒRGFгҖҒSVMгҖҒRNNз®—жі•дҪңдёәеҹәзЎҖеӯҰд№ еҷЁеқҮеҸ–еҫ—дәҶиҫғвҫјзҡ„FDRгҖӮ
    • FARпјҡеӣӣз§Қдј з»ҹзҡ„ ML вҪ…жі•йғҪдёҚиғҪеңЁе°‘ж•°зЈҒзӣҳдёҠеҗҢж—¶жҸҗдҫӣвҫјFDR е’ҢдҪҺ FARгҖӮдј з»ҹ ML вҪ…жі•еҜјиҮҙзҡ„йў„жөӢжҖ§иғҪе·®пјҢвҪҶжі•еҮҸе°‘вҪ¬ж Үеҹҹдёӯе°‘ж•°зЈҒзӣҳж•°жҚ®йӣҶдёҺжәҗеҹҹдёӯеӨҡж•°зЈҒзӣҳж•°жҚ®йӣҶд№Ӣй—ҙзҡ„еҲҶеёғе·®ејӮгҖӮ
    • F-ScoreпјҡTLDFP зҡ„ F-Score иҝңвҫјдәҺе…¶д»–дёҚеҗҢзҡ„дј з»ҹMLвҪ…жі•пјҲ9 еҖҚпјү
    • AUC-ROCпјҡTLDFP зҡ„ AUC-ROC жӣІзәҝйғҪйқ иҝ‘е·ҰдёҠв»ҶпјҢеӣӣз§Қдј з»ҹзҡ„MLвҪ…жі•иҺ·еҫ—дәҶиҫғдҪҺзҡ„ AUC еҖј

  2. дёҺе…¶д»–еҗҢж ·дҪҝз”ЁиҝҒз§»еӯҰд№ з®—жі•зҡ„жЁЎеһӢдҪңеҜ№жҜ”пјҲSSDB, TLBNпјү

    • иҷҪз„¶SSDBеҢ№й…ҚдәҶжәҗеҹҹе’ҢвҪ¬ж Үеҹҹзҡ„еҲҶеёғпјҢдҪҶе®ғеҸӘжҳҜеҜ№жәҗеҹҹдёӯзҡ„и§ӮеҜҹиҝӣвҫҸдәҶжҺ’еәҸпјҢвҪҪTLDFPеҜ№жҜҸдёӘи§ӮеҜҹиҝӣвҫҸдәҶжӣҙжңүж•Ҳзҡ„жқғйҮҚи°ғж•ҙгҖӮ

    • TLBNпјҡеӨҡжәҗеҹҹиҝҒз§»еӯҰд№ пјҢTLDFPпјҡеҚ•жәҗеҹҹиҝҒз§»еӯҰд№ гҖӮ

      еҲҶжһҗKLDзҡ„еҖјпјҡTLDFPдјҡйҖүжӢ©жәҗеҹҹдёӯKLDеҖјжңҖе°Ҹзҡ„пјҢеҚіе·®ејӮжңҖе°Ҹзҡ„

дјҳи¶ҠжҖ§

  • еҜ№дәҺзӣёеҗҢзҡ„ж•°жҚ®йӣҶпјҢTLDFPеҸҜд»Ҙжңүж•Ҳең°и§ЈеҶіе°‘ж•°зЈҒзӣҳж•…йҡңйў„жөӢй—®йўҳпјҢе…¶йў„жөӢжҖ§иғҪвҪҗдј з»ҹMLвҪ…жі•е’Ңе…¶д»–дёӨз§ҚиҝҒз§»еӯҰд№ вҪ…жі•иҰҒеҘҪеҫ—еӨҡгҖӮ

  • TLDFPдёҚд»…жҸҗдҫӣвҫј FDRгҖҒF-Score е’ҢAUCпјҢеҗҢж—¶иҝҳжҳҫвҪ°еҮәзӣёеҪ“дҪҺзҡ„ FARгҖӮ

    вҮ’ TLDFP з®—жі•иғҪеӨҹеҲ©вҪӨе°‘йҮҸж Үи®°зҡ„вҪ¬ж ҮзЈҒзӣҳж•°жҚ®е»әвҪҙжәҗе’ҢвҪ¬ж ҮзЈҒзӣҳжЁЎеһӢд№Ӣй—ҙзҡ„иҒ”зі»пјҢеҮҸе°‘дәҶжәҗеҹҹе’ҢвҪ¬ж Үеҹҹд№Ӣй—ҙзҡ„ж•°жҚ®еҲҶеёғе·®ејӮпјҢдҪҝжЁЎеһӢеҫ—еҲ°иүҜеҘҪзҡ„и®ӯз»ғгҖӮ

е…¶д»–з¬”и®°

  • SMART

    • иҮӘжҲ‘зӣ‘жҺ§гҖҒеҲҶжһҗе’ҢжҠҘе‘ҠпјҢеұһжҖ§еҢ…еҗ«зқҖзЈҒзӣҳйҖҗжёҗйҖҖеҢ–е’ҢеҸҜиғҪеҚҙжҳҫеҫ—дҝЎжҒҜгҖӮиӢҘеұһжҖ§зҡ„еҖји¶…иҝҮзӣёеә”зҡ„йў„е®ҡд№үйҳҲеҖјеҲҷзЈҒзӣҳе°ҶиҮӘеҠЁжҠҘиӯҰгҖӮ

    • зјәзӮ№пјҡFDRпјҡ3-10%пјҢFARпјҡ0.1% вҮ’ дёҚеӨҹзҗҶжғіпјҢж•…йҡңжЈҖжөӢзҺҮеӨӘдҪҺпјҢеӨӘдҝқе®Ҳ

    • SMARTеұһжҖ§зҡ„з»„жҲҗ

      • ID
      • normalized value (1~253)
      • raw valueпјҡеҺҹе§ӢеҖј
      • thresholdпјҡзЈҒзӣҳеҸ‘еҮәж•…йҡңиӯҰжҠҘзҡ„йҳҲеҖј
      • worstпјҡз»ҷе®ҡеұһжҖ§зҡ„жңҖдҪҺжҲ–жңҖе·®еҖј

    • вјӨи§„жЁЎSMARTж•°жҚ®йҮҮйӣҶжЎҶжһ¶пјҡжңүж•Ҳи§ЈеҶіжө·йҮҸеӯҳеӮЁе’ҢжЈҖзҙўйҡҫзҡ„й—®йўҳ
